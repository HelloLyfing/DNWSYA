#再也不要看到你(Do Not Wanna See You Again)

Chrome扩展 for [Startup News][s-n-link]`内容过滤器`，可以根据`用户`、`网站`、`域名`以及`标题(RegEx available)`对页面内容进行过滤和重排

## 关于标题
其实原来的页面按钮打算叫"再也不要看到你"的，鉴于有些人可能受不了过于琼瑶(Men Sao)的名称，所以还是把按钮名称改得中规中矩了一些，叫"清凉一下"
